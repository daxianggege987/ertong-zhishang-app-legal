# 儿童智商测试 · 隐私政策与技术支持（静态页）

本仓库**仅**托管 App Store 所需的简体中文 **`privacy.html`** 与 **`support.html`**，通过 GitHub Pages 对外提供固定 HTTPS 链接。不包含 iOS 应用源代码。

## 页面文件

| 说明       | 路径 |
|------------|------|
| 隐私政策   | `docs/privacy.html` |
| 技术支持   | `docs/support.html` |
| 上架文案草稿 | `docs/app-store-copy.md` |

启用 GitHub Pages（**Source: main 分支 `/docs` 目录**）后，线上地址一般为：

- `https://<用户名>.github.io/<仓库名>/privacy.html`
- `https://<用户名>.github.io/<仓库名>/support.html`

请将上述链接填入 App Store Connect 的「隐私政策 URL」与「技术支持 URL」。

## 修订

修改 `docs/` 下 HTML 后推送至 `main`，数分钟内 Pages 会更新；若遇缓存可无痕窗口重试。
